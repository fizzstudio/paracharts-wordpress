Manage data sets and display them as charts in WordPress.

== Description ==

Allows you to manage data sets via a spreadsheet interface or via CSV import and present that data in an accessible data visualization.

To contribute, report issues, or make feature requests use [Github](https://github.com/fizzstudios/paracharts-wordpress).

== Installation ==

1. Put the `paracharts` directory into your plugins directory
2. Click 'Activate' in the Plugins admin panel
3. Adjust the ParaCharts Settings to your preference
	- WordPress Admin -> Charts -> Settings
